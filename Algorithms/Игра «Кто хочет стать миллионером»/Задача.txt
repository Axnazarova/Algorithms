Напишите игру «Кто хочет стать миллионером».
Задайте пользователю три вопроса с 4 вариантами ответа на любую тематику.
В случае правильного ответа на все вопросы выводится «Вы победили», в случае неверного ответа на вопрос — «Вы проиграли».

1) Напишите функцию, задающую вопрос и проверяющую ответ.
Функция принимает в качестве параметров вопрос, варианты ответов, генерирует сообщение и возвращает логическое значение, демонстрирующее, верный ли ответ выбрал игрок.

2) Задайте вопросы и проверьте их правильность.
Придумайте и задайте вопросы, используя написанную функцию.

3) Добавьте подсказку «50 на 50».
В каждом из вопросов при вводе пользователем «50/50» выводится идентичное окно, в котором вместо 4 вариантов ответа лишь 2.
При попытке ввести «50/50» второй раз выводится окно с надписью «Вы уже использовали подсказку».

4) Добавьте индикатор прогресса.
В начале второго и третьего пунктов должно появляться окно с надписью «Верно! Ваш выигрыш составляет: Х», где Х — 100 для второго пункта и 500 — для третьего.

5) Добавьте возможность забрать выигрыш вместо ответа.
При закрытии любого из окон с полем для ввода в 1–3 пунктах выводит сообщение: «Вы правильно ответили на Х вопросов из 3 и Ваш выигрыш составил Y», где X — 0 для первого пункта, 1 для второго и 2 для третьего, а Y — 0 для первого пункта, 100 для второго и 500 для третьего.
